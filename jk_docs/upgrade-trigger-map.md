# Workflow Ops — “Why Upgrade” Trigger Map

## Free → Pro
**Trigger events**
- User attempts second snapshot
- User clicks Restore
- User promotes workflow
- Drift detected repeatedly

**Message**
“Protect your workflows with automatic snapshots and instant restore.”

---

## Pro → Agency
**Trigger events**
- User connects second environment
- User runs promotions repeatedly
- User invites another team member
- Drift occurs after promotion

**Message**
“Add pipelines, approvals, and drift management for team-safe delivery.”

---

## Agency → Enterprise
**Trigger events**
- User asks for SLA/TTL
- User exports audit data
- User manages multiple teams/tenants
- Repeated drift incidents

**Message**
“Enforce governance with policies, automation, and audit guarantees.”

---

## Rule
Upsells appear only at moments of *felt risk*, never randomly.
